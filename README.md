# tut-pandas

A collection of **Pandas** data manipulation practice notebooks — covering core operations for data wrangling, exploration, and transformation.

---

## What Is Covered

| Topic | Description |
|---|---|
| DataFrame basics | Creating, reading, and inspecting DataFrames |
| Selecting data | `.loc`, `.iloc`, boolean indexing, column selection |
| Filtering | Conditional filtering, `query()`, `isin()` |
| Grouping & aggregation | `groupby()`, `agg()`, `pivot_table()` |
| Merging & joining | `merge()`, `concat()`, join types |
| Handling missing data | `isnull()`, `dropna()`, `fillna()` |
| String operations | `str.upper()`, `str.contains()`, regex |
| Sorting | `sort_values()`, `sort_index()` |
| Apply functions | `apply()`, `map()`, `lambda` |
| Date/time handling | `pd.to_datetime()`, date components |
